# Washing Days

Predicting the ideal times to dry clothes outside using meteorological data

# Live Demo

https://wshelley.github.io/washing-days.txt

Updating every hour.

# Example Usage:
```
pip3 install -r requirements.txt
python3 washing-days.py --lon -3.35 --lat 55.75
```
